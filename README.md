# High-low Counter
Simple Expected-Value and Card counter built for the discord Unbelievaboat higher or lower game
## Installation
For linux g++:
```
g++ @buildGPP.rsp
./app.out
```

For windows cl.exe:
```
cl.exe "@buildCLexe.rsp"
./app.exe
```
## Usage

1. Build and run the program
2. Run the program
3. Start a new game of higher or lower, or input the cards from your active games starting from when the deck was full
4. Input the card you have
5. Read the Expected EVs and percent chance of winning for High, Low, and Same, and the Recommended choice
6. Choose one of the options, most likely the recommended choice
7. Input the dealer's card
8. Repeat from 4-7 until satisfied.

Note: If the program's card count and your card count is off, you most likely have an input error.
